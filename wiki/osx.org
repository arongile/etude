*** Navi - Help

This is so good. Much better than ~tldr~ or ~man~

**** Install

#+BEGIN_SRC shell :results output silent :cache no :eval no
brew install navi

# install terminal widget Ctrl-G to activate
echo "eval \"$(navi widget bash)\"" >> ~/.bash_custom
#+END_SRC

**** Usage

#+BEGIN_SRC shell :results output silent :cache no :eval no
navi
